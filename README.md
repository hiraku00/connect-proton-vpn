# Proton VPN Japan Watcher (macOS)

Proton VPN の無料プランを利用して、日本（Japan）のサーバーに接続されるまで自動で「サーバー切り替え」を試行し続ける監視スクリプトです。

## 特徴

- **自動リトライ**: 日本以外の国に接続された場合、自動的にサーバーの切り替えを実行します。
- **制限の自動回避**: 無料プランの「10分間制限」が発生した場合、自動的にVPNを切断して10分間待機し、解除後に再試行します。
- **スマート検知**: アプリ上のタイマー表示（09:59など）を読み取り、無駄な待ち時間なく制限モードへ移行します。
- **リアルタイム表示**: コンソール上で次の試行までの残り時間をカウントダウン表示します。
- **自動停止**: 日本のサーバーに接続された瞬間に、監視を終了します。

## 前提条件

- **OS**: macOS
- **アプリ**: [Proton VPN GUI アプリ](https://protonvpn.com/download) が `/Applications/ProtonVPN.app` にインストールされていること
- **ツール**:
    - `curl` (IP判定用)
    - `python3` (JSONパース用)
- **権限**: 実行するターミナル（Terminal.app, iTerm2, VSCodeなど）に**「アクセシビリティ」の操作権限**が必要です。
    - 初回実行時にポップアップが出た場合は「許可」してください。
    - または、`システム設定 > プライバシーとセキュリティ > アクセシビリティ` から手動で追加してください。

## 使い方

1. **スクリプトの準備**
   ```bash
   chmod +x ./proton_japan_watch.sh
   ```

2. **アプリの起動**
   Proton VPN アプリを起動しておいてください（ログイン済みであること）。

3. **監視の開始**
   ```bash
   ./proton_japan_watch.sh
   ```

## 動作フロー

1. **初期チェック**: 未接続の場合は「Quick Connect」を試行します。
2. **判定**: 現在の国が「日本」なら即座に終了します。
3. **切り替え**: 日本以外なら「Change server」をクリックします。
4. **待機（通常）**: 接続を安定させるため45秒間待機します（カウントダウン表示）。
5. **制限検知**: もし45秒待ってもボタンが有効にならない、またはタイマーが表示されている場合は、VPNを切断して10分間待機します。
6. **再開**: 制限解除後、自動的に接続を再開して日本を探し続けます。

## 設定

スクリプト内の以下の変数を書き換えることで、待機時間を調整できます。

- `BASE_WAIT`: 通常の試行間隔（デフォルト 45秒）
- `LONG_WAIT`: 制限発生時の待機時間（デフォルト 600秒）

## 免責事項

このスクリプトはUI操作を自動化するものであり、アプリのアップデート等によりボタン名や構造が変わると動作しなくなる可能性があります。

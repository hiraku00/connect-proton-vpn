# Proton VPN Japan Watcher (macOS)

Proton VPN の無料プランを利用して、日本（Japan）のサーバーに接続されるまで自動切り替えを試行し続けるツールです。

## 主な機能

- **スマート待機ロジック**:
  - アプリ上のタイマー（MM:SS）を自動解析し、実際の制限時間に基づいた正確な待機時間を算出。
  - 短い制限時間（60秒未満）の場合は接続を維持したまま待機し、オーバーヘッドを削減。
- **リアルタイム表示**:
  - コンソール上で残り時間を同一行にてカウントダウン表示。ログを汚さず状況を把握可能。
- **自動制限回避**:
  - 長い制限時間が発生した場合は自動的にVPNを切断して待機し、解除後に再接続を試行。
- **自動停止**:
  - 日本のサーバーへの接続を確認した瞬間にプロセスを正常終了。

## 前提条件

- **OS**: macOS
- **アプリ**: [Proton VPN GUI アプリ](https://protonvpn.com/download) が `/Applications/ProtonVPN.app` にインストールされていること
- **ツール**:
  - `curl` (IP判定用)
  - `python3` (JSON解析用)
- **権限**: 実行するターミナル等に**「アクセシビリティ」の操作権限**が必要です。

## セットアップ

```bash
# 実行権限の付与
chmod +x ./proton_japan_watch.sh
```

## 使い方

Proton VPN アプリを起動し、ログインした状態で実行してください。

```bash
./proton_japan_watch.sh
```

## 動作仕様

1. **初期接続**: 未接続時は「Quick Connect」を試行。
2. **国判定**: IP APIを用いて現在の接続国を確認。日本なら終了。
3. **サーバー切り替え**: 日本以外なら「Change server」を実行。
4. **タイマー解析**: アプリUIをスキャンし、制限タイマーの有無と残り時間を取得。
5. **待機戦略**:
   - **60秒未満**: 接続を維持してカウントダウン。
   - **60秒以上**: VPNを切断して待機。
6. **再試行**: 待機終了後、自動的に次の接続試行へ移行。

## 設定

スクリプト内の以下の変数を書き換えることで調整可能です。

- `BASE_WAIT`: 通常の試行間隔（デフォルト 45秒）
- `DISCONNECT_THRESHOLD`: 切断を実行する閾値（デフォルト 60秒）

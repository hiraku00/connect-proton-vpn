# Proton VPN Japan Watcher (macOS)

Proton VPN の無料プランを利用して、日本（Japan）のサーバーに接続されるまで自動切り替えを試行し続けるツールです。

## 主な機能

- **スマート待機ロジック**:
  - アプリ上のタイマー（MM:SS）を自動解析し、正確な残り時間を算出。
  - **条件付き接続維持**: 待ち時間が短い（デフォルト60秒未満）場合は接続を維持したまま待機し、無駄な再接続を回避。
- **リアルタイム表示**:
  - コンソール上で残り時間を同一行にてカウントダウン表示。ログを汚さずに状況を把握可能。
- **自動制限回避**:
  - 長い制限時間（10分等）が発生した場合は自動的にVPNを切断して待機し、解除後に再起動。
- **詳細なカスタマイズ**:
  - 各種待機時間やタイムアウト設定を、マジックナンバーを使わず変数で一括管理。

## 前提条件

- **OS**: macOS
- **アプリ**: [Proton VPN GUI アプリ](https://protonvpn.com/download) がインストールされていること
- **ツール**: `curl`, `python3`
- **権限**: ターミナル等に「アクセシビリティ」の操作権限が必要です。

## セットアップと実行

```bash
chmod +x ./proton_japan_watch.sh
./proton_japan_watch.sh
```

## カスタマイズ（環境変数/スクリプト冒頭）

スクリプト冒頭の以下の変数を書き換えることで挙動を調整できます。

| 変数名 | デフォルト | 内容 |
| :--- | :--- | :--- |
| `BASE_WAIT` | `45` | 通常のリトライ間隔（秒） |
| `LONG_WAIT` | `600` | タイマー未検知時の最大制限待ち（秒） |
| `DISCONNECT_THRESHOLD` | `60` | これ以上の待ち時間ならVPNを切断する閾値（秒） |
| `TIMER_BUFFER` | `2` | アプリタイマーに対する予備時間（秒） |
| `INITIAL_WAIT` | `5` | 起動直後の安定待ち時間（秒） |
| `CONNECT_WAIT` | `10` | サーバー切り替え後の反映待ち（秒） |
| `RECONNECT_WAIT` | `15` | 切断からの再復帰待ち（秒） |
| `AS_TIMEOUT` | `5` | AppleScript操作のタイムアウト（秒） |
